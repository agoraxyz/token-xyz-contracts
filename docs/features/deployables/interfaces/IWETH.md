# IWETH

Interface for WETH.



## Functions
### deposit
```solidity
  function deposit(
  ) external
```
Deposit ether to get wrapped ether.



### withdraw
```solidity
  function withdraw(
    uint256 wad
  ) external
```
Withdraw wrapped ether to get ether.


#### Parameters:
| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
|`wad` | uint256 | The amount of ether to withdraw.
